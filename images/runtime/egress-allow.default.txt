# Monoceros runtime — default egress allowlist.
#
# One hostname per line. Lines starting with `#` and blank lines are
# ignored. Hostnames are resolved at container start; the resulting A
# records become explicit ACCEPT rules in the OUTPUT chain. Anything
# not listed here is dropped by the default policy.
#
# IPv6 is blocked unconditionally — modern Docker setups still run IPv4
# only inside the container by default and we don't want to leave a
# parallel egress path open through ip6tables.
#
# Per-solution overrides go into `.monoceros/egress-allow.txt` inside
# the workspace and are appended to this list at runtime.
